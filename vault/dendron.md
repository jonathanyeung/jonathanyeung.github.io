## Writing pīnyīn on a Mac

Swap to `ABC Extended` Input Keyboard (Ctrl+Opt+Space to change input keyboard)

Type the keyboard shortcut, then the vowel

| Keyboard | Tone  |
| -------- | ----- |
| ⌥ + a    | 1st ā |
| ⌥ + e    | 2nd é |
| ⌥ + v    | 3rd ǎ |
| ⌥ + `    | 4th à |
