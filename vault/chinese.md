## Writing pīnyīn on a Mac

Swap to `ABC Extended` Input Keyboard (Ctrl+Space to cycle)

Type the keyboard shortcut, then the vowel

| Keyboard | Tone  |
| -------- | ----- |
| ⌥ + a    | 1st ā |
| ⌥ + e    | 2nd é |
| ⌥ + v    | 3rd ǎ |
| ⌥ + `    | 4th à |

Ctrl + Opt + Space : Change Input Keyboard
